# ManageIcas

Add Interchain Account to or remove from the treasury.

## Bulk import format

This is relevant when bulk importing actions, as described in [this
guide](https://github.com/DA0-DA0/dao-dao-ui/wiki/Bulk-importing-actions).

### Key

`manageIcas`

### Data format

```json
{
  "chainId": "<CHAIN ID">,
  "register": <true | false>
}
```
