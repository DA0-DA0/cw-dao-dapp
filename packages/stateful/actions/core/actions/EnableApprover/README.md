# EnableApprover

Enable an approver for an existing proposal module.

## Bulk import format

This is relevant when bulk importing actions, as described in [this
guide](https://github.com/DA0-DA0/dao-dao-ui/wiki/Bulk-importing-actions).

### Key

`enableApprover`

### Data format

```json
{
  "approver": "<ADDRESS>"
}
```
